torrent-web-app
===============

A web application (site) suitable for torrent purposes. Just like any other torrent web sites, you can store torrent files and distribute them. Including a Control Panel for administrators and moderators.

System Features:
The following is the list of features:

- User Authentication
- Upload torrents
- Presenting an image of each torrent
- Presenting information of torrent
- torrent categorizing
- Search engine
- Control Panel
- Manage torrents
- Manage Moderators
- Change Password
